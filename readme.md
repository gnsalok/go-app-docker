## go-app-docker

> Create a Docker Container for your Go application. 



#### Build Your image 
> docker build -t go-app-docker .    

#### Check Image details 
> docker image ls 

#### Run you container 
> docker run -p 8080:8080 -tid go-app-docker

#### Check its status 
> docker ps
